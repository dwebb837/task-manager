# Task Manager Application
```
Build a Task Manager Application with advanced features of React Query.
⠀⠀⠀➥ Day 1
⠀⠀⠀⠀⠀⠀• Install React Query and setup useQuery for fetching data.
⠀⠀⠀⠀⠀⠀• Understand the caching mechanism with staleTime and cacheTime.
⠀⠀⠀➥ Day 2
⠀⠀⠀⠀⠀⠀• Study about different ways of using dynamic queries with parameters.
⠀⠀⠀⠀⠀⠀• Implement pagination with useInfiniteQuery and practice background refetching.
⠀⠀⠀➥ Day 3
⠀⠀⠀⠀⠀⠀• Dive into mutations using useMutation for creating and updating data.
⠀⠀⠀⠀⠀⠀• Implement error handling and optimistics updates for a better user experience.
⠀⠀⠀➥ Day 4
⠀⠀⠀⠀⠀⠀• Explore query invalidation and query dependencies.
⠀⠀⠀⠀⠀⠀• Implement features with background polling for real-time data.
⠀⠀⠀➥ Day 5
⠀⠀⠀⠀⠀⠀• Optimize performance with query batching.
⠀⠀⠀⠀⠀⠀• Integrate React Query with global state management and investigate how SSR could be used with React Query.
```
